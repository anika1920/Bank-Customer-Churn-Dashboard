# Bank Customer Churn Analysis Dashboard

## 📌 Project Overview

This project analyzes customer churn behavior in the banking sector using Excel. The dashboard provides key insights into customer attrition patterns and helps identify factors that influence customer churn.

The objective of this project is to:

* Understand customer churn trends.
* Identify high-risk customer segments.
* Analyze customer demographics and banking behavior.
* Support data-driven retention strategies.

---

## 📊 Dashboard Features

### Key Performance Indicators (KPIs)

* Total Customers
* Total Churned Customers
* Churn Rate
* Average Credit Score
* Average Account Balance
* Average Age
* Average Tenure
* High-Value Customers

### Analysis Dimensions

* Churn by Gender
* Churn by Age Group
* Churn by Geography
* Churn by Number of Products
* Churn by Credit Card Ownership
* Churn by Active Membership Status
* Churn by Tenure
* Balance Distribution Analysis

## 📈 Key Insights

* Customers with lower activity levels exhibit higher churn rates.
* Certain geographic regions show significantly higher customer attrition.
* Customers with fewer banking products are more likely to churn.
* Active members tend to have better retention rates.
* Customer age and account balance influence churn behavior.

---

## 🎯 Business Recommendations

* Improve engagement programs for inactive customers.
* Develop retention campaigns for high-risk segments.
* Encourage customers to adopt multiple banking products.
* Offer personalized services to valuable customers.
* Monitor churn indicators regularly using dashboards.
